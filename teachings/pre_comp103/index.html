CanSat Data Payload Brief

The data from CanSat to the ground station is sent in a string format: CSV (Comma Separated Value). In this CanSat testing, the data is of the following:

CS:03,32,13.100200,100.500600,32.59,1013.25,0.00,42.00,3.45,

The data is separated by comma ",". Each value between commas represents one data. For instance:

Position Data (Unit) Examples
1 CanSat ID CS:03
2 Packet counter 32
3 GPS Latitude (deg) 13.100200
4 GPS Longitude (deg) 100.500600
5 Temperature (deg C) 32.59
6 Barometric Pressure (hPa) 1013.25
7 Altitude from SSL (m) 0.00
8 Humidity (% RH) 42.00
9 Battery Voltage (V) 3.45

Using PuTTY to monitor Serial devices (Windows)

  1. Install PuTTY
  2. Install CH340 Driver
  3. Connect the Serial device
  4. Open cmd/terminal/commandline
  5. Type "MODE" in the cmd, the name of device should be COMn, e.g., COM6.
  6. Open PuTTY
  7. On the left pane, click "Serial"
  8. Set Serial line to COMn
  9. Set baudrate to 9600
